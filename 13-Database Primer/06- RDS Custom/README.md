# RDS Custom

## Understanding the Limitation

By default, RDS does not offer any access to the underlying OS where
database engine is running.
For use-cases were customization is required, this can be a limiting factor.

## RDS Custom

RDS Custom makes it possible for you as a database administrator to access
and customize your database environment and operating system.

Supported: Just Microsoft SQL Server

# Benefits of RDS Custom

Install third-party applications.
Modify file systems to support legacy applications.
Apply custom database patches or modify OS packages on your RDS Custom
DB instances.
